# Alliance University RAG System

This project implements a Retrieval Augmented Generation (RAG) system
that answers questions about Alliance University using:

- Website data
- PDF documents
- CrewAI agents
- ChromaDB vector store
- Sentence Transformers embeddings
- RAGAS evaluation

## Features

- Multi-source RAG
- Agent based routing
- Automatic website indexing
- PDF knowledge extraction
- Evaluation using RAGAS
- Gradio interface
